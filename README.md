# Project Title

Memory game implementation

## Running implementation

You can find the running implementation in following link: https://umuthatipoglu.github.io/memory-game/

### Prerequisites

* Node version > 6.9.1
* npm version: 3.10.8

### Running implementation locally

* npm install
* npm start
